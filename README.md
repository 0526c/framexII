# FrameX

AI影视创作平台 — 从创意到分镜，一键生成可拍摄的完整方案。

## 技术栈

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion

## 本地开发

```bash
npm install
npm run dev
npm run build
# 静态文件输出到 dist/ 目录
framex/
├── app/           # 页面
├── components/    # 组件
├── lib/           # 工具函数 & 数据
└── types/         # TypeScript 类型
